### Load Saturation Detection

This code capsule accompanies the article: "A Framework for Detecting The Onset of Load Saturation in Capacity-Constrained Systems"

The capsule contains:
- implementation of the proposed Variance Response–Based Saturation Detection (VRSD) framework
- synthetic data reproducing saturation effect
- notebook for synthetic data with all figures in the paper

Due to confidentiality constraints, real operational data for legacy voice network cannot be shared.
